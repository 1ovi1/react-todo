## Запуск проекта

### Клонирование проекта
```shell
git clone https://github.com/1ovi1/react-todo
```
### Установка зависимостей
```shell
npm i
```
### Запуск проекта
```shell
npm run dev
```

## Реализованные возможности

- Просмотр и редактирование задач
- Автоматическая фильтрация задач по статусу выполнения
